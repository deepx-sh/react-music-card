# Music Player Card UI

A modern and interactive **Music Player Card UI** built using **React** and **Tailwind CSS**.  
It features smooth animations, dark mode support, and a premium glassmorphism design.

---

## Features

- Play / Pause functionality (UI state)
- Like (Heart) toggle
- Thumbs Up / Thumbs Down toggle
- Skip Back / Skip Forward icons
- Dark Mode toggle
- 3D hover animation
- Glassmorphism UI
- Fully responsive design

---

## Tech Stack

- **React (Vite)**
- **Tailwind CSS**
- **Lucide React Icons**

## Getting Started

### Clone the repository

```bash
git clone https://github.com/deepx-sh/react-music-card.git
cd react-music-card
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```
